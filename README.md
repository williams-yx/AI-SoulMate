# AI SoulMate

一个面向 AIGC 与 3D 打印教育与创作场景的开源平台。  
支持文生图、文生 3D、图生 3D、社区展示与 3D 打印工作流，帮助团队快速搭建从创意到实物的完整链路。

## 项目亮点

- AIGC 创作能力：文生图、文生 3D、图生 3D
- 3D 结果管理：作品发布、社区互动、历史记录管理
- 打印业务能力：下单、任务分发、状态跟踪、收货闭环

## 关于我们

模灵科技(北京)有限公司开放自研 AIGC+3D 打印平台「AI SoulMate」，面向教学与创作提供文生图、文生 3D、图生 3D 与打印能力；与高校合作开展培训课程与教学资源，并面向师生普及 AI 生成 3D 实操与行业应用。我们致力于打造从兴趣培养到订单驱动的 AIGC+3D 打印贯通式教育解决方案。

官方网站：`www.aiinfomate.com`

## 核心能力

### 1) AI 造梦工作台

- 文生图：输入提示词快速生成创意图像
- 文生 3D：支持纹理/白模等生成模式
- 图生 3D：支持图片驱动 3D 生成与参数调节
- 本地预览：支持常见 3D 文件加载查看

### 2) 社区与内容生态

- 作品发布与展示
- 点赞、评论、下载等互动能力
- 作品详情页 3D 预览

### 3) 3D 打印闭环

- 从创作结果一键下单
- 打印任务状态流转可视化
- 物流信息与确认收货闭环

### 4) 用户与积分系统

- 账号/手机/邮箱/OAuth 多方式登录
- 身份绑定与账号合并机制
- 免费积分、兑换积分、付费积分分轨管理

## 技术栈

- 前端：Vue 3 + TypeScript + Vite + Pinia
- 后端：FastAPI + PostgreSQL + Redis
- 部署：Docker Compose
- 对象存储/外部服务：按环境变量配置（如 OSS、第三方 AI 服务）

## 快速开始

### 环境准备

- Docker
- Docker Compose

### 启动项目

```bash
docker-compose up -d --build
```

默认访问地址（以本地配置为准）：

- 前端：`http://localhost:8080`
- 后端 API 文档：`http://localhost:3000/docs`

### 常用命令

```bash
# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```
